# borrarTodo

[androidJvm]\
abstract suspend fun [borrarTodo](borrar-todo.md)()

Elimina todos los registros de la tabla de estudiantes.

Útil para realizar una limpieza completa del padrón antes de descargar una lista nueva, evitando que queden registros obsoletos de semestres anteriores.
